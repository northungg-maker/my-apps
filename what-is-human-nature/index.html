第1讲：人性是什么

核心定义

人性就是：只要没有代价，人一定会往"更舒服、更有利"的方向滑。

这不是贬义。这是生物进化的默认设定——所有生命都在追求"低成本、高回报"的路径。人之所以"有人性"，不是因为人特别坏，而是因为人特别会算账。

三个维度拆解

维度一：趋利——人会自动靠近有利的选择
有好处就上，这不是贪婪，是本能。问题在于：当好处是短期的、私人的，代价是长期的、别人的——人性就成了问题。
维度二：避害——人会自动远离有风险的选择
有代价就躲，这不是懦弱，是生存策略。问题在于：当"不作为"的代价被延迟时，人默认选"不作为"——拖延的本质。
维度三：短期偏好——人永远高估眼前的收益
这是行为经济学最扎实的发现。卡尼曼的前景理论证明：人对损失的感受是收益的2.5倍，对眼前利益的权重远超未来利

四个常见误区

  • 误区1："人性本恶" → 人性不是恶，是懒。懒比恶更危险——因为恶有意识，懒是自动的
  • 误区2："我是好人所以不会" → 情境决定行为，品格只是变量之一。斯坦福监狱实验证明：好人放在坏系统里，也会变坏
  • 误区3："意志力能战胜人性" → 靠意志力对抗人性，失败率90%+。有效的是改变环境，不是改变自己
  • 误区4："理解人性=变得冷漠" → 恰恰相反。理解人性让你不再对人性抱幻想，反而能更理性地善待他人
人性不是问题。不设防的人性才是问题。

第2讲：人性怎么运作

核心循环

人性不是一个"特点"，而是一个自我强化的循环系统：

刺激 → 反应 → 短期收益 → 强化 → 循环↑

循环一旦启动，人就会越来越"像人性"，而不是越来越理性。因为每一次强化都在告诉你："上次这样做很爽，这次也可以。"

学术支撑：三个经典实验

① 米尔格拉姆服从实验（1961）

耶鲁大学心理学家米尔格拉姆于1963年招募40名普通人当"老师"，在实验者指令下对学生施加递增电击（最高450伏，实为假电击）。结果：65%的人施加了最高电击——仅仅因为"穿白大褂的权威让我这么做"。后续重复实验证实这一比例稳定在61%-66%之间。

结论：普通人会在权威指令下做出反人性的事。不是他们坏，是情境太强。
人性教训："我绝不会"这句话，在没有代价的情境下不成立。你以为的底线，只是还没被测试过。

② 斯坦福监狱实验（1971）

津巴多于1971年把24名经过心理测试筛选的健康大学生随机分成"看守"和"囚犯"。原计划14天，第6天被迫终止——看守开始虐待囚犯（剥夺睡眠、强迫脱衣、关禁闭），囚犯出现心理崩溃、被动服从。更关键的是：看守们认为自己的行为"很正常"。

结论：不是坏人造成了坏系统，是坏系统制造了"坏人"。角色的力量远超个人的品格。
人性教训：不要考验人性——因为人性的表现取决于它所在的结构。改变结构，才能改变行为。

③ 损失厌恶实验（卡尼曼 & 特沃斯基，1979）

前景理论的核心发现：人对损失的感受强度是同等收益的约2.5倍。丢100块的痛苦 > 捡100块的快乐的2.5倍。这是卡尼曼因此获得2002年诺贝尔经济学奖的成果。

结论：人性的底层不是"贪"，是"怕"。怕失去的力度远超想得到的力度。
人性教训：想让人行动，让他看到"不行动会失去什么"比"行动能得到什么"有效2.5倍。

关键变量

代价大小

代价是否可见、是否即时。看不见的代价=不存在的代价。人不会为"未来可能的代价"改变当下的行为。

收益可见性

收益是否明确、是否即时。即时收益永远赢过远期收益——行为经济学称之为现时偏误（Present Bias）：人对当下收益的权重系统性高于未来收益，即使未来实际价值更高。

社会规范

周围人怎么做的。人是社会动物，"别人都这样"的暗示力极强——从众效应。

权力不对等

谁掌握规则谁就控制人性。没有制衡的权力=人性脱缰的空间。

边界条件

人性规律在以下条件下最强：

  • 代价不可见或被延迟时
  • 收益即时且具体时
  • 缺乏外部约束（规则、监督、惩罚）时
  • 个体处于压力、疲劳、情绪波动时

人性规律在以下条件下减弱：

  • 代价即时且明确时
  • 有强社会规范约束时
  • 决策经过冷静期（延迟决策）时
  • 有明确的身份认同和价值观锚定时

第3讲：人性在现实中长什么样

场景一：合伙

案例：一起吃苦可以，一起分钱不行

老张和老王是大学同学，一起创业。前三年，两人吃泡面睡办公室，谁也不计较。

第四年公司盈利了。老张负责业务，老王负责技术。老张觉得"客户都是我拉的"，老王觉得"没有我的产品你拿什么卖"。谁也说服不了谁。

没有提前约定的分配规则，两个人从合伙人变成对簿公堂的仇人。

真实案例比比皆是：股权均分（50:50）是创业最常见的陷阱——没有决策权的最终裁定者，每次分歧都是僵局。多位法律从业者指出：合伙纠纷中，"均分股权"的案例占比极高，且几乎都以决裂收场。
关键点：不是他们变了，是情境变了——代价从"一起亏钱"变成了"我要少拿钱"。当收益具体化，人性的自动计算就启动了。没约束的合伙=定时炸弹。

场景二：感情

案例：越付出，越不被珍惜

小陈对女朋友百依百顺：随叫随到、从不拒绝、主动付出。两年后，女朋友越来越冷淡，甚至开始嫌他"没主见"。

小陈不理解：我对你这么好，你为什么不珍惜？

因为在他女朋友的心理账户里，小陈的付出已经变成了"免费的"——免费的资源不会被珍惜，这是人性。
关键点：经济学有一个基本定律：没有成本的东西不会被估值。感情里也一样——你的好如果永远零门槛，它就会贬值到零。

场景三：职场

案例：能力很强但爱偷懒的人

小刘是团队里技术最强的，但也是摸鱼最严重的。领导找他谈心，他说"我会改"——然后继续摸鱼。

领导很困惑：他明明有能力，为什么不发挥？

因为在当前的激励机制下，摸鱼的收益（轻松）是即时的，而努力的收益（可能加薪、可能升职）是延迟的、不确定的。人性在做精确的成本收益计算——只是这个计算的结果和公司期望的不一致。
关键点：能力≠产出。能力是潜力，产出需要激励。不绑定结果的机制=允许人性选择偷懒的机制。

场景四：投资

案例：永远追涨杀跌

2021年基金大热，老赵跟风入场，赚了20%不肯止盈——"还在涨呢"。2022年市场回调，浮亏30%终于扛不住割肉——"再不跑更惨"。

典型的追涨杀跌。行为金融学把这叫处置效应：盈利时不愿止盈（贪婪），亏损时急于止损（恐惧）。

这不是老赵笨。这是人性在金钱面前的默认反应——损失厌恶让人比计划多拿，比计划早逃。
关键点：投资亏损的本质往往不是信息不足，而是人性没被管理。止盈止损规则、定投计划——这些不是"技巧"，是对抗人性的结构性设计。

场景五：社交

案例：真诚换来利用

小林性格直爽，对朋友掏心掏肺。结果发现：有的朋友只在需要帮忙时才出现，从不回报；有的人甚至把他分享的隐私当谈资。

小林开始怀疑：真诚错了吗？

真诚没错。但没有筛选的真诚，就是把无成本的好处开放给所有人——人性的自动计算会筛选出"拿好处不回报"的人。
关键点：真诚是策略，不是义务。什么时候真诚、对谁真诚、真诚到什么程度——这是边界问题，不是品质问题。

第4讲：怎么识别、怎么用

识别清单：10个"人性即将出事"的信号

  1. 没有明确规则：全靠"自觉"和"讲道理"——这等于给人性开了绿灯
  2. 代价不可见：做了没人知道、没人追责——人性会自动计算"无代价=可做"
  3. 收益即时可见：好处马上到手，坏处在远期——短视偏好会让你选错
  4. 对方说"咱讲感情"：用感情替代规则=用感情掩盖不对等
  5. 你开始觉得"不好意思"：不好意思拒绝、不好意思谈钱、不好意思设边界——你已经被人性拿捏了
  6. 单向付出模式：你一直在给，对方一直在拿——没有回馈=消耗
  7. "先这样吧"：拖延规则就是放任人性——所有"以后再说"都是"永远不会"
  8. 有人"例外"：规则对某些人无效——这个结构已经失衡了
  9. 质疑被解读为"不信任"：不允许质疑的结构=人性最容易失控的结构
  10. 你发现自己"也不确定"：边界模糊了，"正常"和"不正常"的分界线消失了

应用策略：别靠人，靠结构

在合伙中：先小人后君子

  • 股权分配、退出机制、决策权——白纸黑字写清楚
  • 不要相信"我们关系好不会出问题"——关系好在亏损时有效，在盈利时失效
  • 设置动态调整机制：贡献和收益绑定，而不是固定不变

在感情中：设置门槛

  • 你的好必须有边界——不是无限供给，而是有条件的付出
  • 延迟满足：不随叫随到，保留自己的空间和时间
  • 稀缺才有价值——不是故意制造稀缺，而是真实地保留自己的重心

在职场中：绑定结果

  • 不要靠"责任心"——责任心在没代价时蒸发
  • 把收入和产出绑定：提成制>固定薪；OKR+奖惩>纯信任
  • 及时反馈：延迟的反馈=无效的反馈

在社交中：设置筛选机制

  • 真诚是策略，不是义务——先观察，再深入
  • 对等测试：你付出一次，看对方是否回应。不回应=结束
  • 设置信息边界：不是所有人都要知道你的全部

风险警示：人性应用的三个坑

⚠️ 坑1：变成"只看利益"的人

理解人性≠放弃善意。善意是有的，但善意要有边界。无边界的善意不是善良，是天真。

⚠️ 坑2：过度控制

设规则≠不信任。规则是让关系可持续的基础，不是对人的否定。"先小人后君子"恰恰是为了保护君子关系。

⚠️ 坑3：拿"人性"当借口

"人性就是这样"不能成为做坏事的理由。理解人性是为了预防，不是为了开脱。作恶的人说"人性如此"——那是在给自己的选择找遮羞布。

3. 伴侣越来越冷淡，你该怎么办？
A多付出，让他感受到你的好
B多沟通，了解他怎么了
C拉开距离，重建自身价值
D忍，感情总有起伏
✓ 稀缺才有价值。付出过量只会让"你的好"继续贬值。重心回到自己身上，你的吸引力才会回来。
✗ 越付出，对方越觉得你"随时都在"——零门槛的好=不需要珍惜的好。
场景：感情
4. 关系里你越来越累，不断付出但对方习以为常。问题出在哪？
A付出还不够
B投入结构失衡——不平衡一定恶化
C对方不懂事
D需要更多沟通
✓ 不平衡不会自动修正——你越补，天平越歪。调整投入结构，不是加更多码。
✗ 更努力只会更累。问题不在投入的"量"，而在投入的"结构"——单方面付出=制造贬值。
场景：职场
5. 员工迟到多次，说下次一定改。你该怎么办？
A再沟通一次
B给他时间改
C设惩罚机制
D相信他的承诺
✓ 人不会因为承诺改变，只会因为成本改变。没有代价的承诺=空话。
✗ "下次一定改"在没有代价的情况下=下次继续迟到。成本才驱动行为。
场景：职场
6. 团队效率下降，大家都在摸鱼。该怎么办？
A多开会提醒
B加强监督
C重设奖惩机制
D换人
✓ 行为是被激励驱动的。摸鱼=当前机制奖励摸鱼。改变机制，行为自动改变。
✗ 开会和监督只能短暂施压。人性的计算不停止——只要摸鱼的收益>努力的收益，人会继续摸鱼。
场景：职场
7. 员工不主动，你鼓励也没用。为什么？
A他态度有问题
B主动性是被激发的，不是被要求的
C需要换人
D多表扬
✓ 主动性=主动的收益>被动。当主动没有额外收益，人自然选择被动。
✗ 主动性不是性格，是对激励的响应。没有激励的"主动"是反人性的。
场景：投资
8. 基金赚了20%，你怎么办？
A继续持有，还能涨
B按预定规则止盈
C加仓
D看心情
✓ 投资规则是反人性的结构设计。"看心情"就是让人性接管——人性的默认选择永远是贪。
✗ "还能涨"是贪婪在说话。没有人知道顶在哪里，但规则可以让你不吃亏。
场景：投资
9. 客户犹豫不下单，根本原因是什么？
A价格太高
B产品不够好
C人默认不做决定——不被推动就不会动
D需要更多解释
✓ 不做决定的收益（安全、不出错）>做决定的收益。制造决策点才能打破默认。
✗ 犹豫不是因为信息不够，而是因为"不做决定"本身是零成本的选择。需要的是推动力，不是更多信息。
场景：社交
10. 别人对你态度变差，最可能的原因是什么？
A反思自己哪里做错了
B你给的空间太大了——态度来自你给的边界
C对方心情不好
D需要更好沟通
✓ 态度是边界的结果。你没设边界=释放"可以这样对我"的信号。
✗ 先别急着反思自己。态度变化往往是对你边界的探试——越退让，对方越得寸进尺。
场景：社交
11. 朋友总找你帮忙但从不回报。你该怎么办？
A再帮一次，也许下次他会回报
B设置边界——没有代价的索取不会停止
C提醒他
D忍了
✓ 没有代价的索取=被鼓励的索取。你每次帮忙都在告诉对方：继续找我。
✗ "下次"永远不会来。当索取没有成本，它只会升级，不会自动停止。
场景：社交
12. 别人占你便宜，你总是忍。结果会怎样？
A对方会良心发现
B不设边界=鼓励继续——占便宜会变成习惯
C对方会感激你的宽容
D关系会慢慢变好
✓ 忍让传递的信号是：这样做没有代价。人性的自动计算会得出结论：继续。
✗ 宽容不会换来感激，只会换来更多的试探。人性对"没有代价"的响应永远是"再来一次"。
综合
13. 你总做错误决策。根本原因是什么？
A不够聪明
B决策机制有问题——错误是系统性产物
C经验不够
D运气不好
✓ 反复犯同样的错=不是你笨，是你的决策系统有bug。改机制，不是改自己。
✗ 智力和经验只是输入，决策机制才是处理器。同样的输入换不同的处理器，结果完全不同。
综合
14. 团队内卷越来越严重，最好的解法是什么？
A劝大家别卷了
B改规则——行为由系统驱动
C批评带头的
D忍
✓ 内卷是规则的结果，不是人的选择。当规则奖励内卷，劝说是无效的。
✗ 劝说和批评只能治标。人性会根据激励重新调整行为——改规则才是治本。
综合
15. "不要考验人性"的真正含义是什么？
A人性不可靠，不要相信任何人
B不要创造让人性失控的条件——用结构代替信任
C人性经不起考验，所以不要测试
✓ 核心不是"信不信人"，而是"需不需要信"。有好的结构，你不需要信任；没有好的结构，信任也没用。
✗ 不是说人不可靠——而是说不要依赖人的可靠性。设计好的结构，让可靠性变得不必要。
深度反思
16. 回想一个你"对人失望"的时刻。是你对人性的理解出了问题，还是你的防线出了问题？
场景：职场
17. 员工拖延交付，每次都差一点。根本原因？
A能力不行
B没有截止成本——拖延没代价就会继续拖
C态度不端正
✓ 拖延的本质是：拖延的成本<按时完成的成本。设截止成本，拖延自动消失。
✗ 拖延不是态度问题，是成本问题。当拖延没有代价，理性的人就会选择拖延。
场景：商务
18. 客户嫌贵，你该怎么办？
A降价
B强化价值感——人买的是感觉
C解释成本
D放弃
✓ "嫌贵"=价值感不够。降价只强化"你本来就不值这个价"的认知。
✗ 降价是最差的策略——它告诉客户"你之前的价格有水分"。强化价值感才对。
场景：商务
19. 用户白嫖不付费，最好的办法？
A教育用户
B优化内容
C提高门槛——免费一定被滥用
✓ 免费的资源=无限的资源。不设门槛=吸引只想白嫖的人。
✗ 教育用户和优化内容都是增量投入，但不解决根本问题：免费的东西不会被珍惜。

一句话精髓

人性不是问题，不设防的人性才是问题。
防的不是人，是那个会自动选最舒服路径的系统。

知识卡片

底层逻辑

人性=自动选择低成本高回报路径。这是进化默认值，不是道德问题。

运作循环

刺激→反应→短期收益→强化→循环。每次"爽"都在加深下次的选择。

核心变量

代价可见性、收益即时性、社会规范、权力不对等。

应对策略

不靠人靠结构：设规则、设边界、绑定结果、设置筛选。
