# 双实例访问方式

## 实例1 - T助理

### TUI
```bash
openclaw tui --url ws://localhost:18789 --token c26b6c2df6d2feab7e1d2c042d9b406275ec7b83dbc5dfbf
```

### Dashboard UI
```
http://localhost:18789
```
Token: `c26b6c2df6d2feab7e1d2c042d9b406275ec7b83dbc5dfbf`

## 快捷启动脚本

在 `~/.zshrc` 添加别名：

```bash
alias tui-t='openclaw tui --url ws://localhost:18789 --token c26b6c2df6d2feab7e1d2c042d9b406275ec7b83dbc5dfbf'
alias tui-xm='openclaw tui --url ws://localhost:19789 --token xiaomeng_2026_a1b2c3d4e5f6'
```

之后直接输入 `tui-t` 或 `tui-xm` 即可启动。
