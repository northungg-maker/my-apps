2

撰写文案（助理协助）

告诉我方向和主题，我会协助撰写完整文案，按30秒结构：

3秒钩子（抓注意力）
10秒展开（核心观点）
10秒反转（颠覆认知）
5秒金句（收尾）
3

拆分3个版本

每篇文案拆成3个版本，用于A/B测试：

提问版：以问题开头
结论版：直接甩结论
反常识版：颠覆认知
视频脚本自动存入 iCloud/openclaw/视频脚本/，命名：YYYY-MM-DD_HH-MM_方向_标题.md

数据收集节奏

📊 收集时间表
发布当天记录初始数据
Day 1-3每天记录
Day 4-7隔天记录
Day 7+每周复盘
2

录入数据

数据助手 → 数据录入 → 填写播放/评论/收藏/点赞

系统自动判定：

✅ 评论率≥3% 或 收藏率≥2% → 有效内容
🔥 播放≥1万 或 评论率≥5% → 爆款
3

收集评论

数据助手 → 评论收集 → 分类粘贴

💬 评论分类说明
🔴 需求（有痛点）用户有问题要解决
🟢 认可点赞、共鸣、感谢
⚪ 中立（无意义）无实质内容
🟠 质疑反对、挑战、追问
分类后点「导出评论」可复制给助理或ChatGPT分析
