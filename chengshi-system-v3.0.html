      📊 评分标准（V3.0 核心维度）

      动力系统：目标感、自驱力、行动方向

      分数行为表现（可直接对照判断）
      0分无任何目标，完全被动生活，所有行动都来自外部驱动，毫无自我意愿
      1分偶尔有模糊想法，但从不转化为行动，三分钟热度，说过就忘
      2分有短期目标但不稳定，需要外部督促才能推进，遇困难容易放弃
      3分有较清晰目标，能自我驱动完成中等难度任务，遇到挫折能基本坚持
      4分有长期目标并持续行动，挫折下仍能调整推进，自我驱动稳定
      5分目标清晰可量化，行动完全自驱，形成目标-行动-反馈闭环，持续迭代优化

      认知系统：逻辑判断、归因方式、世界模型

      分数行为表现（可直接对照判断）
      0分逻辑混乱，无法做基本判断，因果倒置，完全无法理解复杂关系
      1分经常错误归因，习惯性外归因（都是别人的错），世界模型严重偏离现实
      2分能做简单判断，但复杂情况容易混淆，归因不稳定，有时合理有时偏激
      3分能做合理判断，基本能正确归因，理解一般复杂度的因果关系
      4分逻辑清晰，归因平衡，能理解复杂因果关系并做出有效预测
      5分高度理性，归因精准，能预测复杂系统演化，世界模型高度贴合现实

      情绪系统：稳定性、自我调节、抗压能力

      分数行为表现（可直接对照判断）
      0分情绪完全失控，稍有刺激就崩溃，严重依赖他人安抚，无法独立面对压力
      1分情绪极易波动，恢复极慢，外界刺激会产生长时间负面影响
      2分情绪波动明显，需要较长时间恢复，压力下表现不稳定
      3分情绪相对稳定，能自我调节，压力下能基本保持正常运转
      4分情绪稳定，自我调节能力强，压力下仍能高效运作
      5分情绪高度稳定，独立应对重大压力，快速恢复并从中学习，情绪成为助力

      行为系统：执行力、持续性、落地能力

      分数行为表现（可直接对照判断）
      0分完全不执行，所有事情停留在想法，无一件事能落地，长期严重拖延
      1分极少执行，长期拖延，只有在外部强制下才偶有行动
      2分需要外部监督才能完成任务，执行力不稳定，经常半途而废
      3分能完成大多数任务，但不够稳定，复杂任务容易掉链子
      4分大多数任务能按时完成，执行力稳定，复杂任务基本能落地
      5分持续高效执行，复杂任务也能完美落地，形成执行-反馈-优化闭环

      资源系统：识别、调配、优化能力

      分数行为表现（可直接对照判断）
      0分完全无法识别资源，对身边资源视而不见，更谈不上利用
      1分能识别极少量显性资源，使用方式粗糙低效，经常浪费资源
      2分能识别部分资源，调配能力有限，偶有优化但不持续
      3分资源识别较好，能基本调配利用，偶尔能优化资源组合
      4分高效识别调配资源，持续优化使用方式，资源利用率高
      5分极高资源敏感度，资源最大化利用，形成资源-成果-新资源正向循环
      ⚠️ 底线规则（V3.0 新增）：
      • 任一核心系统评分为 0 → 系统存在严重缺陷，无法正常运作
      • 行为系统评分为 0 → 动力链完全断裂，后续所有环节失效
      • ≥2个核心系统评分为 0 → 判定为「系统失效型」
      • 评分 ≤2 分 → 必须生成风险转译，不可跳过
