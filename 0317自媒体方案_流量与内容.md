# 0317自媒体方案：流量机制 + 内容结构 + 互动设计

> **IP：清醒道人**  
> **定位：专门拆穿别人活法的人**  
> **受众：小微创业者/个体赚钱者为主，打工人为辅**

---

## 一、平台算法机制详解

### 1.1 核心认知：平台不分发"内容"，只分发"用户反应"

**流量公式：**
```
推荐量 = 基础流量池 × 完播率权重 × 互动率权重 × 停留时长权重
```

**指标优先级（通用）：**
1. 完播率（是否看完）
2. 停留时长（看了多久）
3. 互动率（点赞+评论+收藏+转发）/播放量
4. 复播率（重复观看）

---

### 1.2 抖音算法机制

**核心逻辑：** 完播率 → 停留时长 → 互动率 → 复播率

| 阶段 | 流量池 | 晋级条件 |
|------|--------|----------|
| 冷启动 | 200-500播放 | 完播率>30%，互动率>3% |
| 第一波 | 1000-5000播放 | 完播率>40%，互动率>5% |
| 第二波 | 1万-10万播放 | 完播率>50%，互动率>8% |
| 爆发期 | 10万+播放 | 数据全面优秀+复播率高 |

**关键节点：**
- **前3秒**：生死线。流失率超过70%直接凉。
- **前5秒**：完播率的决定性时刻。
- **结尾3秒**：决定互动率的关键窗口。

**抖音特有的"赛马机制"：**
- 同一时间段，同类内容竞争
- 谁的数据好，谁拿到流量
- **策略：** 避开同赛道高峰期发布

**发布时间建议：**
- 早高峰：7:00-9:00
- 午休：12:00-13:30
- 晚高峰：18:00-21:00
- 深夜党：22:00-24:00（竞争少，但受众有限）

---

### 1.3 B站算法机制

**核心逻辑：** 弹幕活跃度 > 投币/三连 > 完播率 > 播放量

| 指标 | 权重 | 说明 |
|------|------|------|
| 弹幕 | ⭐⭐⭐⭐⭐ | B站灵魂，互动的直观体现 |
| 投币 | ⭐⭐⭐⭐⭐ | 最高认可，直接影响推荐 |
| 三连 | ⭐⭐⭐⭐ | 点赞+投币+收藏的组合 |
| 完播率 | ⭐⭐⭐ | 重要但非决定性 |
| 播放量 | ⭐⭐ | B站不唯流量论 |

**B站用户特点：**
- 习惯长视频（3-15分钟更受欢迎）
- 弹幕文化浓厚
- 对"干货"和"整活"都有需求
- 社区归属感强，愿意为认同的内容投币

**清醒道人在B站的策略调整：**
- 视频时长扩展至3-5分钟
- 增加知识密度（B站用户爱学习）
- 设计"可发弹幕的节点"（如：懂的打个1）
- 结尾求三连，但要有仪式感

---

### 1.4 快手算法机制

**核心逻辑：** 互动率 > 完播率 > 停留时长

**快手特点：**
- **老铁文化：** 粉丝粘性强，信任度高
- **私域权重高：** 粉丝观看占权重
- **互动优先：** 评论、点赞比完播更重要

| 指标 | 权重 |
|------|------|
| 互动率 | ⭐⭐⭐⭐⭐ |
| 粉丝观看比例 | ⭐⭐⭐⭐ |
| 完播率 | ⭐⭐⭐ |
| 私信/分享 | ⭐⭐⭐ |

**快手的"信任电商"逻辑：**
- 粉丝信任你，才会买单
- 高互动率 = 高信任度
- **策略：** 多回复评论，建立老铁关系

---

### 1.5 流量阈值与商业转化

**各平台商业曝光阈值（估算）：**

| 平台 | 播放量门槛 | 互动量门槛 | 商业价值 |
|------|------------|------------|----------|
| 抖音 | 10万+/条 | 点赞5000+ | 品牌方开始关注 |
| B站 | 5万+/条 | 三连1000+ | 接广告有议价权 |
| 快手 | 3万+/条 | 评论500+ | 老铁带货基础 |

**转化标准定义：**

| 行为 | 用户心理 | 商业价值 |
|------|----------|----------|
| 播放 | 好奇/路过 | 低（曝光） |
| 完播 | 有兴趣 | 中（认知） |
| 点赞 | 认同/随手 | 中低（好感） |
| 评论 | 有话说/想参与 | 中高（互动） |
| 收藏 | 有用/以后看 | 高（需求） |
| 转发 | 替我说话/分享给他人 | 很高（传播） |
| 关注 | 期待更多 | 最高（用户资产） |

## 二、30秒视频结构模板（秒级拆解）

### 2.1 通用模板

```
┌─────────────────────────────────────────────────────────┐
│  0-3秒   │ 打脸钩子     │ 否定认知，让人停下来          │
├─────────────────────────────────────────────────────────┤
│  3-10秒  │ 揭穿真相     │ 指出行为本质                  │
├─────────────────────────────────────────────────────────┤
│  10-20秒 │ 上升概念     │ 提出一个"局"的概念            │
├─────────────────────────────────────────────────────────┤
│  20-25秒 │ 收割金句     │ 一句狠话总结                  │
├─────────────────────────────────────────────────────────┤
│  25-30秒 │ 引爆互动     │ 逼评论                        │
└─────────────────────────────────────────────────────────┘
```

---

### 2.2 完整脚本模板

**标题：** 《你以为你在努力，其实你在给别人打工》

```
【0-3秒：打脸钩子】

画面：黑白特写 + 文字浮现
音频：停顿0.5秒，然后用沉重语气开场
台词："你以为你在努力，其实你在帮别人数钱。"

情绪目标：否定认知，制造冲突感

---
【3-10秒：揭穿真相】

画面：切到正常色调，正脸镜头
音频：中速，语气坚定
台词："每天早起，挤地铁，干到半夜，一年下来存了几万块。
然后呢？你老板换了新车，你房东涨了房租。
你以为是努力的问题，其实是位置的问题。"

情绪目标：让用户"对号入座"

---
【10-20秒：上升概念】

画面：加入图形元素或简单动画
音频：放慢，强调关键词
台词："听好了，这是『打工者困境』这个局。
你在别人的系统里，用时间换钱。
系统升级了，工资涨5%；
你老了十年，购买力降50%。
重点来了——你的努力，只够养活你的廉租房梦想。"

情绪目标：建立认知框架

---
【20-25秒：收割金句】

画面：定格画面 + 金句文字浮现
音频：最慢，最有力量
台词："你可以感动自己，但市场不会感动。"

情绪目标：给出结论，制造情绪高点

---
【25-30秒：引爆互动】

画面：切回正脸，直视镜头
音频：挑衅语气
台词："承认吧，你是不是也在这个局里？
评论区打个『在』，我看看有多少人。"

情绪目标：引导评论，制造参与感
```

---

### 2.3 视频节奏控制

| 时间段 | 画面节奏 | 音频节奏 | 文字节奏 |
|--------|----------|----------|----------|
| 0-3秒 | 快速切换或定格 | 强停顿+重音 | 大字闪现 |
| 3-10秒 | 中速切换 | 正常语速 | 关键词高亮 |
| 10-20秒 | 稳定画面 | 放慢强调 | 概念文字 |
| 20-25秒 | 定格特写 | 最慢最终 | 金句大字 |
| 25-30秒 | 切近景 | 加速收尾 | 互动引导 |

## 三、评论/点赞/收藏/转发触发机制

### 3.1 评论触发三模型

#### 模型A：对立站队型

**原理：** 让用户选择立场，制造阵营对立

**模板：**
```
1. "你觉得这个人，是努力还是傻？评论区站队。"
2. "努力重要，还是选择重要？打出来。"
3. "如果你是他，你会怎么选？A还是B？"
4. "这个观点你认同吗？认同打1，不认同打2。"
```

**适用场景：** 有明确立场的话题，如：打工vs创业、躺平vs卷

**注意：** 话题要有争议性，但不能触碰红线

---

#### 模型B：自我暴露型

**原理：** 触发用户的"我也有过"心理，引导分享

**模板：**
```
1. "你干过最傻的投资是什么？评论区说出来让我们笑笑。"
2. "你被割过多少次韭菜？报个数。"
3. "你现在最想逃离的是什么？评论区匿名说。"
4. "你离躺平还有多远？还差多少钱？"
```

**适用场景：** 容易引发共鸣的话题，如：踩坑经历、焦虑分享

**技巧：** 
- 用"让我们笑笑"减轻用户的尴尬感
- 用"匿名"降低暴露门槛

---

#### 模型C：认知羞辱型（慎用）

**原理：** 激发用户的"我不是这种人"心理

**模板：**
```
1. "承认吧，你就是不敢。"（等评论反驳）
2. "看懂的人，不需要解释。看不懂的，你开心就好。"
3. "如果这条让你不舒服，说明说到你了。"
```

**风险：** 容易引发反感，需配合IP人设使用

**适用场景：** 极度自信的观点、强硬立场

**使用原则：**
- 只在结尾使用，不要全程攻击
- 针对行为，不针对人
- 保持"清醒道人"的冷眼旁观姿态，不要变成喷子

---

### 3.2 点赞触发机制

**点赞的心理：** "说到我心里了" + "我认可" + "顺手点一下"

**触发条件：**

| 触发点 | 操作方法 | 示例 |
|--------|----------|------|
| 情绪共鸣 | 说中用户的心声 | "你是不是也这样想过……" |
| 替我表达 | 用户想说但没说出的话 | "你领导永远不懂的那种心累" |
| 金句共识 | 一个简单的真理 | "努力不是货币，结果才是" |
| 视觉冲击 | 画面有记忆点 | 惊讶表情 + 触目数据 |
| 音乐配合 | BGM在情绪高点 | 金句出现时音乐推向高潮 |

**视觉引导：**
- 金句出现时，画面左下角点赞图标轻微跳动
- 评论提示"觉得对的点个赞"

---

### 3.3 收藏触发机制

**收藏的心理：** "这个有用，以后会用到"

**触发条件：**

| 类型 | 操作 | 示例 |
|------|------|------|
| 方法论 | 给出可执行的步骤 | "记住这三步……" |
| 数据信息 | 给出有价值的数字 | "2024年副业赚钱的3个方向……" |
| 认知框架 | 给出思维模型 | "判断一个项目能不能做，看这三点……" |
| 避坑指南 | 给出负面清单 | "这些副业，千万别碰……" |
| 清单体 | 结构化信息 | "搞钱路上的5个坑、4条路、3个原则" |

**话术引导：**
- "建议收藏，以后用到可以翻出来"
- "这条值得反复看，先收藏"
- "截图发给自己，这几点记下来"

---

### 3.4 转发触发机制

**转发的心理：** 
1. 替我说话→我说不出来的，你帮我表达
2. 显得我懂→转发显得我深刻
3. 讽刺别人→暗戳戳@某人
4. 分享价值→让朋友也看看

**触发设计：**

| 心理 | 设计方法 | 示例话题 |
|------|----------|----------|
| 替我表达 | 说中打工人的心声 | "给老板看看，什么叫加班" |
| 显得我懂 | 有逼格的观点 | "穷人思维vs富人思维" |
| 讽刺别人 | 可以@人的内容 | "@你那个天天加班的朋友" |
| 讨论话题 | 引发朋友圈讨论 | "你觉得努力重要还是选择重要？" |

**可@性设计：**
- 结尾加："转给你那位还没清醒的朋友"
- 弹幕/评论引导："@你朋友来看"

---

### 3.5 关注触发机制

**关注的心理：** "我想看更多这样的内容"

**触发条件：**
1. 内容有价值（有用）
2. 人设有魅力（认同）
3. 有系列感（期待）
4. 有互动感（被看见）

**话术模板：**
```
1. "下期讲《如何找到自己的第一桶金》，想看的点个关注。"
2. "我是清醒道人，专门拆穿那些忽悠你的东西。"
3. "每天一条，帮你清醒一点。关注不迷路。"
4. "想让我拆穿什么？评论区说，点赞最高的下期讲。"
```

## 四、口头禅完整设计

### 4.2 开场口头禅（身份标志）

**功能：** 
1. 让观众3秒内知道是谁、什么风格
2. 形成品牌记忆
3. 引发期待

**设计原则：**
- 必须有"反转"或"否定"的调性
- 必须暗示"我要说真话了"
- 朗朗上口，容易记住

**正式版：**

| 编号 | 口头禅 | 气质 | 使用场景 |
|------|--------|------|----------|
| A1 | "清醒一下。" | 冷静、简短 | 开场固定标志 |
| A2 | "你以为你在xx，其实你在xx。" | 打脸式 | 准备揭穿某事 |
| A3 | "大部分人一辈子，都没看懂这个局。" | 居高临下 | 讲认知类话题 |
| A4 | "来，说个扎心的。" | 朋友式 | 准备讲真相 |
| A5 | "今天不熬鸡汤，只说实话。" | 对立式 | 区别于其他账号 |

**开场组合示例：**
```
示例1：
"清醒一下。你以为你在努力，其实你在帮别人数钱。"

示例2：
"来，说个扎心的。你存的那些钱，可能连自由都买不起。"

示例3：
"大部分人一辈子，都没看懂打工这个局。今天帮你拆一下。"
```

---

### 4.3 中段口头禅（节奏控制）

**功能：**
1. 控制视频节奏
2. 提示"重点来了"
3. 增加记忆点

**设计原则：**
- 简短有力
- 声音要有变化
- 形成条件反射

**正式版：**

| 编号 | 口头禅 | 语调 | 使用场景 |
|------|--------|------|----------|
| B1 | "重点来了。" | 放慢、加重 | 准备给核心观点 |
| B2 | "很多人卡在这一步。" | 叹息式 | 指出盲区 |
| B3 | "你发现问题了吗？" | 提问式 | 引导思考 |
| B4 | "听好了。" | 命令式 | 给关键结论 |
| B5 | "说到这儿，很多人不乐意听了。" | 预判式 | 准备说逆耳的话 |

**中段插入示例：**
```
"……你每天都在干这个。重点是来了——你干的事，能被替代吗？"

"……很多人卡在这一步：把忙碌当成努力。"

"……你发现问题了吗？你的收入，和你的努力，根本不在一条线上。"
```

### 4.4 结尾口头禅（收口+互动）

**功能：**
1. 情绪收口
2. 制造记忆点
3. 引导互动

**设计原则：**
- 有"收尾感"
- 有"留白"，让观众有反应空间
- 可以单独使用，也可以组合

**正式版：**

| 编号 | 口头禅 | 气质 | 使用场景 |
|------|--------|------|----------|
| C1 | "这就是差距。" | 肯定式 | 给对比后 |
| C2 | "你可以不认同，但现实不会变。" | 强硬式 | 有争议的观点后 |
| C3 | "看懂的人，不需要解释。" | 筛选式 | 高认知内容后 |
| C4 | "你自己想想，是不是这个事儿。" | 引导式 | 让用户自己得出结论 |
| C5 | "清醒道人，下期见。" | 品牌式 | 常规收尾 |

**互动引导话术：**
| 编号 | 话术 | 类型 |
|------|------|------|
| D1 | "认同的，评论区打个1。" | 站队型 |
| D2 | "你呢？评论区说说。" | 自我暴露型 |
| D3 | "转给你那个还没清醒的人。" | 转发型 |
| D4 | "觉得有用的，点个收藏，以后能翻出来看。" | 收藏型 |
| D5 | "想让我下期讲什么？评论区说。" | 参与型 |

---

### 4.5 口头禅使用规范

**原则：**
1. **开场固定用A2/A3/A4**，形成记忆
2. **每条视频必须用1-2个中段口头禅**，控制节奏
3. **结尾必须有互动引导**，D类话术必用
4. **同一口头禅不要连续使用超过3条视频**

**配速建议：**

| 视频类型 | 开场 | 中段 | 结尾 |
|----------|------|------|------|
| 打脸型 | A2+A1 | B1+B2 | C1+D1 |
| 讲理型 | A3 | B3+B5 | C4+D2 |
| 干货型 | A4 | B1+B4 | C2+D4 |
| 情感型 | A1 | B2+B3 | C4+D3 |

## 五、彩蛋和情绪设计具体方案

### 5.1 什么是彩蛋？

**定义：** 视频中隐藏的小惊喜，用于提高完播率和复播率

**作用：**
1. **提高完播率** → 用户为了找彩蛋看完
2. **提高复播率** → 用户二刷三刷确认
3. **提高评论量** → 用户讨论彩蛋
4. **强化人设** → 彩蛋风格=内容风格

---

### 5.2 彩蛋类型与操作

#### 类型A：视觉彩蛋

**定义：** 画面中隐藏的细节

**操作方式：**

| 彩蛋 | 操作方法 | 示例 |
|------|----------|------|
| 背景标语 | 背景白板/墙上写梗 | "努力 = 0，结果 = 1" |
| 悄悄话字幕 | 小字一闪而过 | "工资涨5%通胀涨15%" |
| 道具植入 | 用有梗的道具 | 拿个写着"韭菜"的袋子 |
| 表情包 | 画面角落放表情 | 辛苦说到老板时放狗头 |
| 倒计时戳 | 画面某个数字在变 | 账户余额在递减 |

**执行细节：**
- 不要大，要小（用户要找的那种）
- 不要全，要缺（让用户在评论区说"发现了没"）

---

#### 类型B：听觉彩蛋

**定义：** 声音中的隐藏细节

**操作方式：**

| 彩蛋 | 操作方法 | 示例 |
|------|----------|------|
| BGM梗 | 用有梗的BGM | 讲股市亏钱时放《天台见》 |
| 表情音效 | 配合画面的声音 | 说到老板放"哇哦"音效 |
| 背景音 | 远处的声音彩蛋 | 背景放股票暴跌的新闻播报 |
| 自言自语 | 结尾快速念一句话 | 快语速念"别被割了别被割了" |

---

#### 类型C：互动彩蛋

**定义：** 需要用户参与才能发现的

**操作方式：**

| 彩蛋 | 操作方法 | 示例 |
|------|----------|------|
| 评论区暗号 | 评论区置顶"暗号是这个" | 置顶："懂的懂了" |
| 回复彩蛋 | 作者回复带梗 | 回复评论："我知道你看完了" |
| 下期预告 | 视频最后一秒写下一期主题 | 下期：《副业的真相》 |
| 隐藏观点 | 说一半，评论区补全 | "完整版我在评论区说了" |

---

#### 类型D：连续彩蛋

**定义：** 跨视频的，形成"追剧感"

**操作方式：**

| 彩蛋 | 操作方法 | 示例 |
|------|----------|------|
| 道具连续 | 每期出现同一道具 | 每期桌上有罐"韭菜酱" |
| 口头禅连续 | 每期说同一句话 | 每期结尾都"你就是不敢" |
| 剧情连续 | 上期问题+下期答案 | 上期问+下期答 |
| 数字连续 | 视频编号有规律 | #001 #002 #003 |

---

### 5.3 彩蛋执行清单

**每期视频检查：**
- [ ] 至少有1个视觉彩蛋
- [ ] BGM有梗或反差
- [ ] 结尾有预告或暗号
- [ ] 道具/背景有可用于"找茬"的细节

---

### 5.4 情绪设计方法论

**核心逻辑：** 内容 = 情绪载体，触达情绪 = 触达行动

**情绪路径：**
```
触发情绪 → 情绪高点 → 情绪转化 → 情绪收口
```

---

### 5.5 五种核心情绪触发设计

#### 情绪一：焦虑

**触发场景：** 
- 揭露某种无法逃避的困境
- 指出用户的错误认知
- 预警未来的风险

**操作方式：**

| 环节 | 操作 | 示例 |
|------|------|------|
| 开场 | 定义焦虑对象 | "你每天努力，但你确定方向对吗？" |
| 中段 | 具象化焦虑 | "一年过去，存款没变，年龄涨一岁" |
| 结尾 | 缓解+引导 | "下次讲怎么破局"（给希望但不给全） |

**注意：** 焦虑后必须给希望/方向，否则用户会逃避

---

#### 情绪二：愤怒

**触发场景：**
- 揭露不公现象
- 指出用户被利用的真相
- 打破某种美好幻想

**操作方式：**

| 环节 | 操作 | 示例 |
|------|------|------|
| 开场 | 定义愤怒对象 | "你以为你在加班，其实你在被榨干" |
| 中段 | 强化愤怒 | "老板换了车，你换了眼霜" |
| 结尾 | 转化为行动 | "转发给同样被榨的人" |

**注意：** 愤怒是容易传播的情绪，但需控制火候

---

#### 情绪三：认同

**触发场景：**
- 说中用户心声
- 替用户表达无法表达的东西
- 站在用户这一边

**操作方式：**

| 环节 | 操作 | 示例 |
|------|------|------|
| 开场 | 说中现状 | "你是不是也有这种感觉……" |
| 中段 | 深入共情 | "说不出来，但就是累" |
| 结尾 | 形成共识 | "我们都懂的" |

---

#### 情绪四：好奇

**触发场景：**
- 提出反直觉观点
- 暗示有秘密
- 制造悬念

**操作方式：**

| 环节 | 操作 | 示例 |
|------|------|------|
| 开场 | 制造信息差 | "99%的人都不知道的是……" |
| 中段 | 一步步揭示 | "真正原因是……" |
| 结尾 | 留钩子 | "但还有一点更重要，下期讲" |

---

#### 情绪五：希望

**触发场景：**
- 焦虑/愤怒之后的收口
- 给出方向
- 让用户看到可能

**操作方式：**

| 环节 | 操作 | 示例 |
|------|------|------|
| 开场 | 承认困境 | "你被困住，但不是没出路" |
| 中段 | 指出方向 | "这几个方向，比你现在更靠谱" |
| 结尾 | 给期待 | "下期详细讲" |

---

### 5.6 每期情绪设计清单

**检查表：**
- [ ] 本期主情绪是什么？（焦虑/愤怒/认同/好奇/希望）
- [ ] 情绪触发点在几秒？（应在3-10秒区间）
- [ ] 情绪高点在几秒？（应在20-25秒区间）
- [ ] 情绪收口是否给了希望/方向？
- [ ] 结尾是否有下期钩子？

---

### 5.7 解决什么问题？

**清醒道人的核心价值：** 不是解压，而是"帮用户看清局"

**具体解决的问题：**

| 问题类型 | 具体例子 | 内容价值 |
|----------|----------|----------|
| 认知困境 | 努力了还是穷 | 帮用户识别"局" |
| 选择困境 | 打工还是创业 | 帮用户看清"代价" |
| 风险预警 | 这个项目能不能做 | 帮用户识别"坑" |
| 心态困境 | 焦虑/迷茫 | 帮用户找到"定位" |

**最终目的：** 让用户看完视频后，少踩一个坑、多一分清醒

### 6.2 热点筛选标准

**可用热点的标准：**

| 条件 | 说明 |
|------|------|
| 相关性 | 与清醒道人定位（拆穿、认知、搞钱、职场）相关 |
| 情绪度 | 能引发焦虑/愤怒/认同情绪的话题 |
| 时效性 | 24小时内爆发的话题，最迟48小时 |
| 安全性 | 不触碰政治、宗教、性别等敏感话题 |
| 可延展 | 能上升到"局"的层面，不是纯八卦 |

**筛选判断流程：**
```
1. 这个话题与"清醒道人"定位相关吗？ → Yes继续，No放弃
2. 这个话题能引发什么情绪？ → 有情绪继续，太平淡放弃
3. 这个话题安全吗？ → 安全继续，敏感放弃
4. 这个话题能上升到"局"吗？ → 能继续，不能放弃
5. 现在还热吗？ → 热继续，凉了放弃
```

---

### 6.3 热点利用模式

#### 模式A：借势表态

**操作：** 针对热点事件，给出清醒道人视角的观点

**模板：**
```
开场："xx这事火了，说几句你们不一定爱听的。"
中段：拆解事件本质 → "这不是xx的问题，是xx这个局的问题"
结尾：上升到普遍规律
```

**示例：**
- 热点：某公司裁员
- 内容：《裁员潮来了，你准备好了吗？》
- 视角：不是同情打工人，而是讲"打工这个局"

---

#### 模式B：蹭热度讲干货

**操作：** 用热点话题作为引子，讲自己的干货

**模板：**
```
开场："最近xx很火，很多人问怎么看。"
过渡："其实更重要的是……"
中段：讲与热点相关的干货内容
```

**示例：**
- 热点：某网红翻车
- 内容：《网红翻车背后的自媒体真相》
- 视角：讲自媒体行业的"坑"

---

#### 模式C：反向追问

**操作：** 针对热点，提出不一样的视角

**模板：**
```
开场："所有人都在说xx，但有没有人想过一个问题……"
中段：提出反直觉观点
结尾：揭示本质
```

**示例：**
- 热点：某副业很火
- 内容：《所有人都在做副业，为什么还是穷？》
- 视角：反向思考，指出副业误区

### 7.6 数据对标标准（第一阶段）

## 九、附录

### 附录A：金句库（可直接使用）

**打工类：**
- 你可以感动自己，但老板不会感动。
- 努力不是货币，结果才是。
- 你在别人的系统里，用时间换钱。

**创业类：**
- 自由的代价，是你承受不住的。
- 创业不是逃离打工，是换了个笼子。
- 你的"副业"，可能只是另一份打工。

**搞钱类：**
- 资产和劳动的收入，有天和地的差距。
- 你以为投资是理财，其实是给庄家送钱。
- 把焦虑变成韭菜的人，比你还懂焦虑。

**心态类：**
- 承认吧，你就是不敢。
- 看懂的人，不需要解释。
- 你可以不认同，但现实不会变。
