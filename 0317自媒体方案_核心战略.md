## 一、用户群体完整画像

### 1.1 主用户：小微创业者/个体赚钱者

**核心特征**

| 维度 | 画像 | 数据支撑 |
|------|------|----------|
| **年龄** | 25-40岁，核心28-35岁 | 创业主力年龄段 |
| **性别** | 男性略多（约55:45） | 商业/认知类内容受众偏好 |
| **职业** | 个体户、小微创业者、自由职业者、副业打工人 | 自己要对结果负责的群体 |
| **地理** | 一线及新一线城市为主，下沉市场有增量 | 杭州、深圳、成都、武汉等创业活跃城市 |
| **学历** | 本科及以上为主 | 认知类内容消费门槛 |
| **收入** | 年收入10-50万（不稳定） | 有付费能力，但并非高净值 |

**核心心理特征**

```
┌─────────────────────────────────────────────────────────┐
│  "我每天都很忙，但不知道方向对不对"                        │
│  "别人看我自由，其实我最焦虑"                              │
│  "努力了，但结果不一定是我的"                              │
└─────────────────────────────────────────────────────────┘
```

**为什么他们是主用户？**

1. **收入不稳定** → 极度焦虑"方向对不对"，愿意为"清醒"付费
2. **自己要对结果负责** → 没人告诉他答案，渴望有人指点
3. **时间碎片化** → 短视频是最佳学习渠道
4. **决策成本低** → 愿意试一试99-999元的课程/咨询
5. **复购潜力大** → 破了一个局，还有下一个局

**用户的共性**

| 共性类型 | 具体表现 | 痛点强度 |
|----------|----------|----------|
| **方向焦虑** | 不确定自己做的对不对，不知道该往哪走 | ⭐⭐⭐⭐⭐ |
| **孤独感** | 没人理解，没人商量，没人告诉他真相 | ⭐⭐⭐⭐⭐ |
| **认知瓶颈** | 努力了但没结果，怀疑自己的认知有问题 | ⭐⭐⭐⭐ |
| **信息过载** | 看了很多内容，越看越迷茫 | ⭐⭐⭐⭐ |
| **身份认同** | 不敢说自己创业，怕被看低 | ⭐⭐⭐ |

**如何找到他们？**

| 渠道 | 他们会在哪里出现？ | 影响力排序 |
|------|-------------------|-----------|
| **抖音** | 关注创业、搞钱、认知提升类账号 | ⭐⭐⭐⭐⭐ |
| **小红书** | 搜索"副业""创业心得""踩坑提醒" | ⭐⭐⭐⭐ |
| **B站** | 看商业故事、创业复盘、认知干货 | ⭐⭐⭐⭐ |
| **微信** | 创业群、副业群、行业社群 | ⭐⭐⭐ |
| **播客** | 听创业故事、商业访谈 | ⭐⭐⭐ |

**他们关注什么内容？**

- "X年创业踩过的坑"
- "我是怎么亏掉第一个100万的"
- "打工和创业的认知差异"
- "普通人翻身的真实路径"
- "别被这些创业鸡汤骗了"

**如何吸引他们产生共鸣和依赖？**

| 吸引方式 | 具体做法 | 让用户产生什么感觉 |
|----------|----------|-------------------|
| **说出心声** | "你表面光鲜，但心里慌得要死" | 被看穿、被理解 |
| **拆穿谎言** | "那些教你赚钱的人，靠的是教你赚钱" | 终于有人说真话 |
| **给方向** | "你焦虑的不是没钱，是不知道对不对" | 原来 problem 是这个 |
| **给方法** | 清醒点，别被忽悠，这样做 | 有人指路了 |
| **持续陪伴** | 每天一个清醒时刻 | 有个清醒的人带着我 |

**消费能力和消费意愿**

| 维度 | 评估 | 说明 |
|------|------|------|
| **消费能力** | 中等偏上 | 有收入，但非高净值 |
| **消费意愿** | **很强** | 为"破局"付费意愿极高 |
| **价格敏感度** | 中等 | 可以接受99-999元产品，超过2000元会犹豫 |
| **决策周期** | 短 | 看到有用立刻买 |

**用户认同、认可、持续关注的理由**

1. **说的对** → 终于有人说出了真相
2. **说得狠** → 不装，不美化，直接戳穿
3. **说得清** → 不绕弯子，结论直接给
4. **持续在** → 每天都有新内容，不掉线
5. **人格魅力** → 清醒道人这个人格让人想关注

### 1.2 辅用户：一线城市认知焦虑打工人

**核心特征**

| 维度 | 画像 |
|------|------|
| **年龄** | 25-35岁 |
| **职业** | 大厂员工、传统行业白领、体制内人员 |
| **特征** | 有稳定收入，但内心焦虑，想改变但不敢动 |
| **痛点** | 工作无意义、晋升无望、想创业不敢、不知道人生怎么过 |

**为什么是辅用户？**

- **流量放大器**：人数多，活跃度高，能放大传播
- **潜在付费群体**：部分会转化为付费用户
- **社交传播者**：喜欢转发分享，带来二次传播

**与主用户的区别**

| 维度 | 主用户（小微创业者） | 辅用户（认知焦虑打工人） |
|------|---------------------|------------------------|
| 付费意愿 | 高 | 中 |
| 付费能力 | 中高 | 中 |
| 行动性 | 强（已经在行动） | 弱（还在观望） |
| 内容需求 | 破局方法 | 情绪共鸣+认知提升 |
| 变现价值 | 高（主力人群） | 中（流量放大） |

## 二、IP名字定版

### 2.1 定版结果

# 清醒道人

### 2.2 名字拆解

| 元素 | 含义 | 效果 |
|------|------|------|
| **清醒** | 态度、能力、痛点击中 | 用户想要的状态 |
| **道人** | 人格、角色、神秘感 | 可装疯卖傻、可正经、可狂 |
| **组合** | 一个能让人清醒的人 | 有人格、有态度、有记忆点 |

### 2.3 为什么是"清醒道人"？

**对标参考：**

| 参考 | 特点 | 我们借鉴了什么 |
|------|------|----------------|
| 半佛仙人 | 有"仙人"，有人格 | 用"道人"，同样有人格感 |
| 半佛仙人 | "半佛"有态度 | "清醒"是态度，更直接 |
| 万能公 | 名字大、敢吹 | "道人"有狂气 |
| 三板斧 | 简单好记 | 两个字好记 |

**必须满足的条件：**

| 条件 | 清醒道人是否满足 | 说明 |
|------|------------------|------|
| 好记 | ✅ | 4个字，有节奏感 |
| 张扬 | ✅ | "道人"意味着狂一点没关系 |
| 能整活 | ✅ | 可以装疯卖傻、玩梗 |
| 有爽感 | ✅ | "清醒"击中痛点 |
| 品类感 | ✅ | 一听就是认知类内容 |

**可延展性：**

- 口头禅："清醒点"
- 系列内容："道人清醒时刻"""道人破局指南"
- 二创空间：粉丝可以叫"道友"、"局中人"
- 人设延展：可以时而装神秘、时而疯狂吐槽

### 2.4 IP人设定型

**人设三层：**

```
┌─────────────────────────────────────────────────┐
│  外壳：道人气质，有点疯癫，说话不装                │
│  内核：清醒的观察者，拆穿虚伪，点破困局            │
│  态度：我就是我，爱听不听，听了就清醒              │
└─────────────────────────────────────────────────┘
```

**语气风格：**

- "我跟你说实话..."
- "清醒点..."
- "你以为你懂了？"
- "我见过太多这样的人了..."

**视觉意向：**

- 头像：可以是一个道人剪影/插画风格
- 色调：偏冷色系（清醒感）+ 金色点缀（智慧感）
- 封面风格：大字标题 + 简洁设计 + 一句狠话

## 四、差异化定位及价值主张

### 4.1 一句话定位

> **清醒道人：专门拆穿别人活法的人**

### 4.2 不是什么

| 我们不是 | 为什么 |
|----------|--------|
| 不是知识博主 | 不教书、不讲理论、不给知识点 |
| 不是商业博主 | 不分析公司、不讲商业模式 |
| 不是治愈博主 | 不安慰你、不给你鸡汤 |
| 不是励志博主 | 不打鸡血、不喊口号 |

### 4.3 是什么

| 我们是 | 说明 |
|--------|------|
| 一面镜子 | 让你看见自己的困局 |
| 一个拆穿者 | 拆穿谎言、拆穿自我欺骗 |
| 一个清醒者 | 保持冷静、给出真相 |
| 一个同路人 | 我也困过，我找到了出口 |

### 4.4 差异化价值主张

**思想卡牌系统**

| 维度 | 说明 |
|------|------|
| **是什么** | 把复杂思想做成卡牌，一个视频=一张卡牌 |
| **系统性体现** | 卡牌有系列、有分类、可收集、可组合 |
| **独特性** | 别人给你知识，我们给你一张"思想卡牌" |
| **流量逻辑** | 卡牌有收集欲、分享欲、二创空间 |

**思想卡牌 vs 传统知识内容：**

| 传统知识内容 | 思想卡牌 |
|-------------|----------|
| 一堆知识点 | 一张可收藏的卡片 |
| 看完就忘 | 收藏、转发、反复看 |
| 系统性弱 | 系列化、可组合 |
| 无互动 | 可以"集卡"、可以分享位置 |

**为什么"把复杂思想做成卡牌"能带来流量和认同？**

| 理由 | 数据/推理支撑 |
|------|--------------|
| 收藏欲 | 收藏率高的内容更容易获得算法推荐 |
| 分享欲 | 卡牌形式便于转发到朋友圈 |
| 二创空间 | 用户可以根据卡牌内容创作自己的解读 |
| 系统感 | 系列内容更容易产生追更行为 |
| 简洁感 | 一张卡牌=一个核心观点，降低认知负担 |

**内容被模仿怎么办？**

| 模仿点 | 我们的护城河 |
|--------|-------------|
| 内容形式（卡牌） | 卡牌可以模仿，但人格无法模仿 |
| 视频结构 | 结构可以复制，但洞察和态度无法复制 |

## 五、IP价值优先级

### 5.1 优先级排序

# 情绪 > 表现形式 > 内容

### 5.2 为什么是这个顺序？

**用户行为路径：**

```
被情绪触动 → 被表现形式吸引 → 记住内容
    ↓              ↓              ↓
  想继续看       觉得有意思      收藏/转发
    ↓              ↓              ↓
  点赞/评论      关注主页        进入私域
```

**详细解释：**

| 优先级 | 元素 | 作用 | 为什么优先 |
|--------|------|------|-----------|
| **1** | 情绪 | 让用户停留、产生共鸣 | 没有情绪触动，用户3秒就走 |
| **2** | 表现形式 | 让用户觉得"有意思" | 形式决定完播率 |
| **3** | 内容 | 让用户获得价值 | 在情绪和形式的包裹下，内容才能被吸收 |

### 5.3 具体执行框架

**情绪优先（前3秒）**

| 情绪类型 | 触发语 | 示例 |
|----------|--------|------|
| 好奇 | "大多数人不知道" | "大多数人不知道，你焦虑的不是没钱" |
| 共鸣 | "你是不是也有这种感觉" | "你是不是也觉得自己很努力，但就是没结果" |
| 惊叹 | "我见过太多这样的人" | "我见过太多这样的人，表面光鲜，心里慌得要死" |
| 不服 | "你以为你懂了？" | "你以为你懂了？你只知道一半" |

**表现形式（全流程）**

| 形式元素 | 作用 |
|----------|------|
| 卡牌视觉 | 一张卡牌=一个观点，强化记忆 |
| 短小精悍 | 30秒以内，节奏紧凑 |
| 金句密度 | 每10秒一个可转发的句子 |
| 口头禅 | "清醒点"强化品牌 |

**内容价值（骨架）**

| 内容要求 | 说明 |
|----------|------|
| 一个核心观点 | 不贪多，一个视频讲清楚一件事 |
| 有逻辑支撑 | 不是鸡汤，有推理 |
| 可操作 | 给方向，不只是讲问题 |

### 5.4 爆款公式

```
爆款 = 强情绪钩子（前3秒）+ 高完播形式（中间）+ 可传播金句（结尾）
```

**具体示例：**

```
[0-3秒] 情绪钩子：
"清醒点，你表面光鲜，但心里慌得要死"
（触发共鸣情绪）

[3-25秒] 核心观点 + 表现形式：
一张思想卡牌：[焦虑的根源不是没钱，是不知道方向对不对]
用30秒讲清楚这张卡牌的内容
（高信息密度 + 卡牌形式）

[25-30秒] 金句收尾：
"困在局里的人，需要的不是安慰，是一面镜子"
（可转发的金句）
```

### 5.5 搞怪+内容=吸引粉丝？

**结论：是的，但要注意度**

| 元素 | 作用 | 注意点 |
|------|------|--------|
| **搞怪** | 吸引注意力、增加趣味性 | 不能搞怪过头，失去信任感 |
| **内容** | 建立信任、提供价值 | 内容必须有干货，不能只是好笑 |
| **组合** | 搞怪负责吸引，内容负责留存 | 搞怪+内容的平衡 |

**清醒道人的搞怪边界：**

| 可以做 | 不能做 |
|--------|--------|
| 装疯卖傻、夸张语气 | 低俗、无底线 |
| 玩梗、调侃 | 失去专业感 |
| 偶尔自黑 | 让人怀疑你的可信度 |

**度：**

> 搞怪到让人觉得"这人有点意思但不傻逼"
