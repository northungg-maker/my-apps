## 发现的问题

**核心问题：`db` 变量未定义**

控制台日志：
```
[DEBUG] finalizeContract called
[DEBUG] currentContract: {...}
[DEBUG] currentContract.mode: dual
[DEBUG] currentContract.status: draft
[DEBUG] 进入场景1
[DEBUG] 双方模式，创建 pending 状态
[DEBUG] 准备写入数据库，roomId: ...
[ERROR] 双方模式创建失败: ReferenceError: db is not defined
```

Firebase 数据库引用 `db` 未初始化，导致无法写入数据。

---

## 解决方案

检查 Firebase 初始化代码，确保 `db` 变量被正确定义。

## 代码追踪

**用户流程：**
1. 选择"双向对赌" → `currentMode = 'dual'`
2. 填写表单 → `submitContract()` → `currentContract.mode = 'dual'`
3. 预览页面点击"确认并签署" → 进入签名页面
4. 输入签名 → `confirmSignature()` → 显示 final-confirm-overlay
5. 点击"确认签署并锁定" → `finalizeContract()`
