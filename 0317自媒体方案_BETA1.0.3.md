# 第一部分：核心战略

## 一、用户群体画像

### 1. 核心用户（付费主力）

| 维度 | 描述 |
|------|------|
| **身份** | 小微创业者 / 个体赚钱者 / 自由职业者 |
| **年龄** | 25-40岁 |
| **特征** | 自己对结果负责、收入不稳定、极度焦虑"方向对不对" |
| **痛点** | 每天很忙但不知道在忙什么、努力了但没结果、想改变但不知道从哪开始 |
| **心理需求** | 需要有人告诉他"真相"，而不是鸡汤安慰 |
| **消费能力** | 有收入，有付费意愿，愿意为"能解决问题"的内容买单 |
| **消费场景** | 深夜焦虑刷手机时、遇到决策困境时、复盘反思时 |

**为什么是他们？**
- 评论动机强：需要表达、需要认可
- 付费意愿强：他们知道"认知差"就是钱
- 传播意愿强：会转发给同病相怜的人

---

### 2. 辅助用户（流量放大器）

| 维度 | 描述 |
|------|------|
| **身份** | 一线城市打工人（有认知焦虑的） |
| **年龄** | 22-35岁 |
| **特征** | 对现状不满、想突破、但还没迈出那一步 |
| **痛点** | 职场天花板、不知道该不该跳槽/创业 |
| **心理需求** | 需要情绪共鸣和方向感 |
| **作用** | 帮你传播、撑流量、但不一定是主力付费人群 |

---

### 3. 禁区用户（排除）

| 类型 | 原因 |
|------|------|
| 学生 | 没钱，痛点不同 |
| 纯娱乐用户 | 不思考，完播但无转化 |
| 佛系人群 | 不焦虑，无需求 |

## 二、IP名字定版

**为什么是它：**
| 要求 | 满足情况 |
|------|----------|
| 明确好记 | ✅ 四个字，朗朗上口 |
| 更张扬 | ✅ "道人"自带态度感 |
| 更狂 | ✅ "清醒"两个字直接怼 |
| 能整活 | ✅ 可以装疯卖傻、阴阳怪气 |
| 要爽 | ✅ 听完有种"被点醒"的感觉 |

**备选：**
- 局中人（更哲学，但爆点略弱）
- 破局客（太普通，没记忆点）

**Slogan：**
> **人生是局，清醒破局**

## 五、IP价值优先级

### 结论：情绪为王 > 表现形式 > 内容

**为什么？**

用户的行为路径：
1. 先被情绪触动 → 停下来看
2. 再被表现形式吸引 → 看完
3. 最后记住一两句金句 → 带走

**爆款公式：**
> 强情绪钩子 + 高完播形式 + 可传播金句

# 第二部分：流量机制 + 内容结构 + 互动设计

## 一、平台算法机制

### 核心逻辑

> **平台不分发"内容"，只分发"用户反应"**

核心指标（按权重排序）：
1. 完播率（最重要）
2. 停留时长
3. 互动率（评论 > 转发 > 点赞）
4. 复播率

---

### 抖音算法

| 维度 | 机制 |
|------|------|
| 核心指标 | 完播率优先 |
| 生死线 | 前3秒决定是否继续推 |
| 赛马机制 | 新视频先推200-500人池，数据好再放大 |
| 商业曝光阈值 | 播放量10万+ 可接广告；50万+ 商业价值明显 |

---

### B站算法

| 维度 | 机制 |
|------|------|
| 核心指标 | 弹幕 + 投币 + 收藏 |
| 特点 | 粉丝粘性强，长视频友好 |
| 商业曝光阈值 | 粉丝10万+ 可接商单；播放10万+ 广告报价显著提升 |

---

### 快手算法

| 维度 | 机制 |
|------|------|
| 核心指标 | 互动率（评论/点赞比高） |
| 特点 | "老铁文化"，真实感优先 |
| 商业曝光阈值 | 播放量5万+ 可接广告 |

---

## 二、30秒视频结构模板

### 标准结构（必须严格执行）

| 秒数 | 任务 | 情绪目标 |
|------|------|----------|
| **0-3秒** | 打脸钩子 | 否定认知，让人停下来 |
| **3-10秒** | 揭穿真相 | 指出行为本质 |
| **10-20秒** | 上升概念 | 提出一个"局"的概念 |
| **20-25秒** | 收割金句 | 一句狠话总结 |
| **25-30秒** | 引爆互动 | 逼评论 |

---

### 完整脚本示例：打工人困境

**【0-3秒】钩子**
> "你以为你在努力，其实你在自我麻痹。"

**【3-10秒】揭穿**
> "你每天加班、学习、考证，看起来很拼。
> 但这些都不会改变你的位置。
> 因为你拼的方向，是别人设计好的。"

**【10-20秒】上升**
> "这个局叫'边角料陷阱'。
> 你一直在做边角料的事——边角料的努力、边角料的成长。
> 真正赚钱的位置，你从来没碰过。"

**【20-25秒】收割**
> "人和人的差距，从来不是努力。
> 是你知不知道自己在哪个局里。"

**【25-30秒】引爆**
> "你现在在哪个局里？评论区说实话。"

---

### 变体模板

| 类型 | 钩子示例 | 概念示例 |
|------|----------|----------|
| **创业版** | "你以为创业是出路，可能是进坑。" | "资源错配陷阱" |
| **职场版** | "你以为跳槽能解决问题？你可能只是换个笼子。" | "隐形天花板" |
| **搞钱版** | "你以为省钱是攒钱？其实它在让你变穷。" | "消费降级陷阱" |

## 三、互动触发机制

### 评论引爆三大模型

| 模型 | 方式 | 示例 | 适用场景 |
|------|------|------|----------|
| **模型A：对立站队** | 给两个选项，让人选 | "努力重要还是选择重要？你选哪个？" | 通用，最强 |
| **模型B：自我暴露** | 让人说出自己的状态 | "你现在月薪多少？评论区说实话。" | 触发参与感 |
| **模型C：认知羞辱** | 刺激不服气的人 | "看不懂的人，还在底层循环。" | 慎用，易招黑 |

---

### 点赞机制

**触发条件：** 情绪认同 → "说到我心里了"

**话术设计：**
> "太真实了"
> "这句话扎心了"
> "我终于有人说出来了"

---

### 收藏机制

**触发条件：** 实用价值 → "以后我还会用到"

**话术设计：**
- 给模型（三点法则）
- 给结构（判断方法）
- 给清单（执行步骤）

---

### 转发机制

**触发条件：** 社交表达 → 替我说话 / 显得我懂 / 讽刺别人

**话术设计：**
> "转发给需要的人"
> "有人应该看看这个"

## 三、盈亏平衡点

### 计算公式

> 销售额 = 流量 × 转化率 × 客单价

### 估算

| 假设 | 数值 |
|------|------|
| 粉丝数 | 30,000人 |
| 转化率 | 2%（保守估计） |
| 客单价 | 50元（入门产品） |
| 人均购买次数 | 1次/年 |

**销售额 = 30,000 × 2% × 50 × 1 = 30,000元**

### 结论

**盈亏平衡点：约3-4万粉丝，年销售额2.5万元**
